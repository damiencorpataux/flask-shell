A small flask example to execute shell commands via a REST API and a Web page.

Install & Usage
-
```sh
git clone ...
cd flask-shell
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt
```

```sh
python3 app.py
open localhost:9999
```
